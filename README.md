# Telegram Bot Tools

Web-based tools for managing Telegram bots.

## Features

### Bot Names Tool
- Updates your bot's name, short description, and long description in multiple languages
- CSV format example:
```
en,My Awesome Bot
es,Mi Bot Increíble
de,Mein Tolles Bot
fr,Mon Bot Génial
```

### Bot Sender Tool
- Sends a message to selected chats from your bot

## Tech Stack
- Vite
- Svelte 5
- TypeScript
- Bootstrap 5
- Vitest, Testing Library, Playwright

## Development
- clone the repo
- `yarn`
- `yarn dev` to run the dev server
- `yarn test` to run tests in watch mode
- `yarn test:ui` to run tests with UI
- `yarn test:e2e` to run e2e tests
